# iOS Security Visibility Dashboard

This project is a senior capstone prototype developed in Xcode using SwiftUI.

## Project Purpose
The app helps iPhone users interpret selected Apple Analytics diagnostic logs by translating technical log indicators into plain-language findings, risk levels, and recommendations.

## Main Features
- Diagnostic log input
- Crash report detection
- Jetsam / memory pressure detection
- Panic log detection
- Watchdog event detection
- Risk scoring
- Plain-language recommendations

## How to Run
1. Download or clone the repository.
2. Open the Xcode project file.
3. Run the app using the iOS Simulator.
4. Paste or import a sample diagnostic log.
5. Press Analyze to view the report.

## Disclaimer
This project is for educational purposes only. It does not perform antivirus scanning or full forensic analysis.
